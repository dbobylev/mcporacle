## Проект mcporacle

## Описание

Проект реализует локальный mcp сервер, который обращается к Oracle DB и возвращает полезную информацию
Будет использоваться в Roo Code

## Технологии

python (oracledb, mcp)

## Ограничения

Хост windows без докерезации
mcp сервер не должен иметь возможность выполнять произвольные query sql, только ограниченный жесто докумментированный функционал в рамках tools

## MVP

1. Подключение к Oracle DB
2. Tool: Предоставить информацию о таблице: описание таблицы и колонок. Реализовать через вызов анонимного блока pl/sql c несколькими запросами к dba view
